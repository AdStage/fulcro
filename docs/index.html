Community

Fulcro is an open-source project that is supported by people like you. If you find it useful, please consider contributions in any of the following forms:

  • Reporting issues, ideally with a reproducible case.
  • Look for things that need testing or additions and do that.
  • Proofreading the website/documentation.
  • A pull request fixing a bug (see CONTRIBUTING).
  • Expansions via a pull request (please check in on Slack first).
  • Donations: help to feed the code monkey!
